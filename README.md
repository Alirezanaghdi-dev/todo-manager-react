# Todo List App

This is a simple **Todo List App** built with **React**.

---

## Features

* Add new tasks
* Mark tasks as completed
* Undo completed tasks
* Delete tasks
* View total, completed, and pending tasks

---

## Technologies Used

* React
* JavaScript
* CSS

---

## How to Run the Project


1. Go to the project folder:

```bash
cd myapp
```

2. Install dependencies:

```bash
npm install
```

3. Start the app:

```bash
npm run dev
```

Open your browser and go to:

```
http://localhost:5173
```

---

## Description

Users can add tasks, complete them, undo completion, and delete tasks. The app also shows simple statistics about tasks.
